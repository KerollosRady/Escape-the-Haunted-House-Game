# Escape the Haunted House Game

Escape the Haunted House Game is a first-person horror adventure where the player must explore a haunted house and its surrounding garden to collect hidden keys and escape through the main gate. The player must solve puzzles, navigate dark environments, and avoid a horror character chasing themâ€”all within a limited time. If the player runs out of time or is caught, the game ends in failure.
